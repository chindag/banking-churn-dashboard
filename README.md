# Project Documentation

## Features
- Comprehensive analysis of banking churn data.
- User-friendly dashboard for visualization.
- Data import and export functionalities.
- Customizable reports and insights.

## Setup Instructions
1. **Clone the repository:**
   ```bash
   git clone https://github.com/chindag/banking-churn-dashboard.git
   ```
2. **Navigate into the project directory:**
   ```bash
   cd banking-churn-dashboard
   ```
3. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
4. **Run the application:**
   ```bash
   python app.py
   ```
